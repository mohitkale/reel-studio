# Roadmap

Public roadmap for Reel Studio. Near-term items are realistic milestones.
Larger ideas live under **Exploratory** and are not promises.

## Near term

- Easier first-run setup (`npm run setup` / `npm run demo`)
- Polished demo project and demo podcast (no API keys required)
- More HyperFrames templates (Apache-2.0 path)
- Better caption editing in the editor
- Export presets for Reels, Shorts, and LinkedIn
- Reliable local voice workflow (Kokoro + VoiceForge docs)
- README product visuals and starter templates
- Template authoring guide for contributors

## Later

- Batch rendering for multiple formats
- Custom template SDK / validation helpers
- Desktop packaging (Electron or equivalent)
- Optional plugin architecture for templates and providers
- Stronger MCP workflows for end-to-end production

## Exploratory (not scheduled)

These may never ship; discuss in [GitHub Discussions](https://github.com/mohitkale/reel-studio/discussions) if interested:

- Real-time collaboration and cloud sync
- Distributed / multi-machine rendering
- Native mobile companion apps
- Real-time voice conversion
- Deep 3D model import pipelines

See also [CONTRIBUTING.md](CONTRIBUTING.md) for how to propose work.
